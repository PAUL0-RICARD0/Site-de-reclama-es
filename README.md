Um sistema simples de registro e acompanhamento de reclamações desenvolvido com HTML, CSS e JavaScript puro.


O projeto tem como objetivo permitir que usuários:
- Registrem reclamações
- Acompanhem o status das solicitações
- Tenham uma interface simples e intuitiva

Tecnologias Utilizadas:
HTML5
CSS3
JavaScript (Vanilla JS)

📂 Estrutura do Projeto
site de reclamações/
│
├── index.html                # Página inicial
├── reclamacao.html           # Página para registrar reclamação
├── acompanhar.html           # Página para acompanhar reclamações
│
├── assets/

│   │
│   └── images/

│       └── fundo.jpg


Funcionalidades
- Cadastro de reclamações
- Consulta/acompanhamento de reclamações
- Interface estilizada com CSS
- Interatividade com JavaScript

Como Executar:

Baixe ou clone este repositório:

git clone <url-do-repositorio>

Acesse a pasta do projeto:

cd site-de-reclamacoes

Abra o arquivo index.html no navegador.
